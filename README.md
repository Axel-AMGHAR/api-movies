# Lancer le projet

- clone projet
- install docker
- run `./vendor/bin/sail up`
- run `./vendor/bin/sail artisan migrate`
- go to http://localhost
- register
- go to api-token in the menu
- generate the token
- use the token to connect to the database
- go to http://localhost/docs to see the api documentation
